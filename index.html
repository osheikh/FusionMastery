Day 1
Parametric Thinking & Fusion Foundations
4-5 hours

Objective: Understand how Fusion works and build fully parametric, predictable models from your FIRST design.

üåÖ Morning: Interface & Navigation Mastery (1.5 hours)

The Problem with Most Tutorials: They rush past navigation. Students then spend weeks fighting the viewport instead of designing.

Core Navigation Skills

OrbitMiddle mouse + dragSee your model from any angle

ZoomScroll wheelGet closer/further
Zoom to FitF or double-click middle mouseLost? This saves you

Exercise 1.1 Navigation Practice

Open a new design. Spend 15 minutes ONLY navigating. Orbit to see the origin planes from every angle. This feels stupid. Do it anyway.

Data Panel & File Management

  • Understand Projects vs Folders
  • CRITICAL: Fusion saves to the cloud. Learn to use "Export" for local .f3d backups
  • Version history ‚Äî right-click any design to see previous versions

‚òÄÔ∏è Midday: Parameters FIRST ‚Äî The Professional Mindset (1 hour)

This is the most important habit you'll build. Professionals create parameters before sketching. Beginners hard-code numbers and regret it later.
Exercise 1.2 Your First Parametric Setup
  1. Create new design
  2. Go to Modify ‚Üí Change Parameters
  3. Click "+" to add User Parameters:
box_length = 60 mm
box_width = 40 mm
box_height = 25 mm
wall_thickness = 2 mm
corner_radius = 5 mm

Click OK ‚Äî these are now available in ANY dimension field

Why This Matters: Later, when your client says "make it 80mm long," you change ONE number. The entire model updates. This is parametric design.

üåÜ Afternoon: Sketching ‚Äî The Foundation of Everything (2.5 hours)

Hard Truth: 80% of your modeling success depends on clean sketches. Garbage sketch = garbage model.

The Golden Rule: Fully Constrained Sketches

  • Black lines = fully constrained (GOOD)
  • Blue lines = under-constrained (BAD for parametric design)
  • Red lines = over-constrained (ERROR)
Exercise 1.3 Parametric Box Sketch
  1. Create new sketch on XY plane (click XY plane in browser, then Create ‚Üí Sketch)
  2. Draw a Center Rectangle starting at the origin
  3. Add dimensions using your parameters: click dimension tool (D), click a line, type box_length instead of a number
  4. Repeat for width using box_width
  5. Add corner fillets: Sketch ‚Üí Fillet, select corners, type corner_radius
  6. Goal: All lines must be BLACK
Exercise 1.4 Extrude Using Parameters
  1. Finish sketch (click "Finish Sketch")
  2. Press E for Extrude
  3. Select your sketch profile
  4. For distance, type box_height (not "25")
  5. Click OK
Exercise 1.5 Verify Parametric Behavior
  1. Go to Modify ‚Üí Change Parameters
  2. Change box_length to 100mm
  3. Click OK
  4. Your model should update automatically. If it doesn't, you hard-coded something.

Day 2
Sketching with Intent & Constraints
5 hours

Objective: Gain complete control over geometry using constraints, construction lines, and origin-based sketching.

üåÖ Morning: Geometric Constraints Deep Dive (2 hours)

Constraints are MORE important than dimensions. A well-constrained sketch with few dimensions is better than a poorly-constrained sketch with many dimensions.

Sketch Constraints (MEMORIZE THESE)

Construction Geometry ‚Äî Your Secret Weapon

Press X to toggle any line to construction (dashed). Construction lines:

  • Don't become part of the final profile
  • Perfect for centerlines, reference angles, symmetry axes
  • Help constrain without adding geometry
Exercise 2.1 Mounting Plate with Parametric Holes 
plate_length = 80 mm
plate_width = 50 mm
plate_thickness = 4 mm
hole_diameter = 5 mm
hole_inset = 8 mm
corner_fillet = 3 mm
  1. Sketch centered rectangle on XY plane using plate_length and plate_width
  2. Draw construction lines from center to corners (for reference)
  3. Add 4 circles for mounting holes
  4. Constrain holes: hole_inset from each edge (use dimension from edge, not absolute position)
  5. Make all holes Equal to each other (select all 4, press E)
  6. Add corner fillets using corner_fillet
  7. Extrude using plate_thickness
Key Learning: The holes are positioned RELATIVE to edges. If you change plate_length, holes move with it.

üåÜ Afternoon: Projecting & Referencing Geometry (1.5 hours)

Project brings existing geometry into your sketch as reference.

Exercise 2.2 Adding Features to Existing Geometry
  1. Take your mounting plate from Exercise 2.1
  2. Create a new sketch on the TOP face of the plate
  3. Use Sketch ‚Üí Project/Include ‚Üí Project to bring the edges into your sketch
  4. Sketch a centered boss (raised cylinder) for a component to sit on
  5. Constrain the boss to be centered using the projected edges
  6. Extrude the boss 6mm

üåô Late Afternoon: Symmetry & Mirror (1.5 hours)

Exercise 2.3 Symmetric Bracket 
bracket_height = 40 mm
bracket_base = 60 mm
bracket_thickness = 5 mm
slot_width = 10 mm
slot_height = 20 mm
  1. Sketch on XZ plane (side view)
  2. Draw a vertical construction line on the Y-axis (this is your mirror line)
  3. Sketch HALF of an L-bracket profile on one side
  4. Use Sketch ‚Üí Mirror to create the other half
  5. Add a slot cutout (for cable routing or weight reduction)
  6. Extrude using bracket_thickness
Why Mirror? Change one side, both sides update. Perfect symmetry guaranteed.

Day 3
Features for Strength & Printability
5 hours

Objective: Learn which modeling features produce strong, printable parts ‚Äî and which create problems.

üåÖ Morning: Essential Modify Tools (2 hours)

ToolUse CaseShortcutPrint Consideration
FilletRound edgesFExternal: aesthetic. Internal: CRITICAL for strength
ChamferAngled edges-Better than fillet on bottom edges (no elephant foot)
ShellHollow out bodies-Reduces material, must maintain wall thickness
DraftAdd angle to walls-Helps with mold release AND print bed adhesion
RibAdd reinforcement-Stronger than thick walls, less material
CRITICAL FOR 3D PRINTING:
  • Bottom edges: Use chamfer (0.5-1mm), not fillet ‚Äî fillets create elephant's foot issues
  • Internal corners: ALWAYS fillet ‚Äî sharp internal corners are stress concentrators and crack points
  • Vertical walls: Consider 1-2¬∞ draft angle for better print quality
Exercise 3.1 Parametric Enclosure 
enclosure_length = 70 mm
enclosure_width = 50 mm
enclosure_height = 30 mm
wall_thickness = 2 mm
bottom_chamfer = 1 mm
top_fillet = 2 mm
internal_fillet = 1.5 mm

Build sequence:

  1. Sketch centered rectangle, extrude to enclosure_height
  2. Shell with wall_thickness, removing TOP face
  3. Add bottom_chamfer to ALL bottom external edges
  4. Add top_fillet to ALL top external edges
  5. Add internal_fillet to ALL internal corners (this is the step most people skip ‚Äî don't)
  6. Use Section Analysis (Inspect ‚Üí Section Analysis) to verify wall thickness

‚òÄÔ∏è Midday: Ribs vs Solid Walls (1.5 hours)

The Amateur Mistake: Making walls thicker for strength.

The Professional Approach: Adding ribs.

Ribs use 70% less material than thick walls while providing equal or better strength.

Exercise 3.2 Ribbed Base Plate 
base_length = 100 mm
base_width = 60 mm
base_thickness = 3 mm
rib_height = 8 mm
rib_thickness = 2 mm
rib_spacing = 15 mm
  1. Create base plate
  2. Sketch rib pattern on bottom face (use Rectangular Pattern in sketch)
  3. Extrude ribs downward using rib_height
  4. Add fillets where ribs meet the base (stress concentration!)
  5. Compare: this is MUCH stronger than a 10mm solid plate

üåÜ Afternoon: Print Orientation Planning (1.5 hours)

Layer lines are your weakest point. Parts break along layer boundaries.
Load DirectionBest Print Orientation
Tension/pullingLayers perpendicular to force
CompressionLayers perpendicular to force
BendingLayers parallel to bend axis
ShearAvoid shear loads on printed parts
Exercise 3.3 Orientation-Aware Hook

Design a wall hook that will hold 5kg. Consider:

  1. Where will the load be applied?
  2. Which direction will the hook want to break?
  3. How should you orient it for printing?
hook_depth = 40 mm
hook_height = 30 mm
hook_thickness = 8 mm
mounting_holes = 4 mm
wall_plate_width = 50 mm
wall_plate_height = 60 mm

Design the hook, then sketch (on paper or in Fusion) how you would orient it on the print bed and WHY.

Day 4
Components, Assemblies & Tolerances
5-6 hours

Objective: Design multi-part models that actually fit together in the real world.

üåÖ Morning: Bodies vs Components ‚Äî The Critical Distinction (1.5 hours)

ConceptWhat It IsWhen to Use
BodyDumb geometry, no relationshipsSingle-piece designs, boolean operations
ComponentSmart container with own origin, timelineMulti-part assemblies, anything that moves
Rule: If it's a separate physical part, it MUST be a component.
Exercise 4.1 Component Structure Setup
  1. Create new design
  2. IMMEDIATELY right-click the top-level component ‚Üí "Create Component"
  3. Name it "Box_Base"
  4. Create another component named "Box_Lid"
  5. Notice: each component has its own origin in the browser
Why This Matters: Components can be moved independently, exported as separate STL files, reused in other designs, and animated with joints.

‚òÄÔ∏è Midday: Tolerances ‚Äî The #1 Reason Parts Don't Fit (2 hours)

Create these as GLOBAL parameters you'll reuse: 
clearance_press = -0.1 mm
clearance_snug = 0.05 mm
clearance_sliding = 0.25 mm
clearance_loose = 0.5 mm
clearance_hinge = 0.4 mm
Fit TypeClearanceUse CaseExample
Press Fit-0.1 to -0.2mmPermanent assemblyBearing into housing
Snug Fit0.0 to 0.1mmFriction hold, removableBattery cover
Sliding Fit0.2 to 0.3mmMoving partsDrawer
Loose Fit0.4mm+Easy assemblyDecorative cap
Hinge/Pivot0.4 to 0.6mmRotationLiving hinge, pin joint
Exercise 4.2 Two-Part Box with Parameterized Clearance 
box_inner_length = 60 mm
box_inner_width = 40 mm
box_inner_height = 25 mm
wall = 2 mm
lid_height = 8 mm
lip_depth = 4 mm
lip_clearance = 0.25 mm  // THIS IS THE KEY

Box Base (Component 1):

  1. Activate Box_Base component
  2. Sketch and extrude outer shell
  3. Shell to create walls
  4. Add internal lip for lid to sit on

Box Lid (Component 2):

  1. Activate Box_Lid component
  2. Design lid to fit INSIDE the lip
  3. Lid dimensions = lip dimensions MINUS lip_clearance on each side
  4. Example: if lip opening is 56x36mm, lid insert is 56 - lip_clearance*2 x 36 - lip_clearance*2

üåÜ Afternoon: Joints & Motion Testing (2 hours)

Joint Types

JointDegrees of FreedomExample
Rigid0Glued parts
Revolute1 rotationHinges, wheels
Slider1 translationDrawers
Cylindrical1 rotation + 1 translationPiston
Pin-Slottranslation (constrained path)Crank mechanism

Exercise 4.3 Hinged Lid
  1. Position lid above box using Move/Copy
  2. Assemble ‚Üí Joint
  3. Select edge on lid, then matching edge on box
  4. Choose "Revolute" joint type
  5. Test: drag the lid ‚Äî it should rotate like a hinge
  6. Use "Motion Study" to animate
Exercise 4.4 Interference Check
  1. With lid closed, go to Inspect ‚Üí Interference
  2. Select both components
  3. Fusion will highlight any overlapping geometry
  4. If interference exists, adjust your clearance parameter and rebuild

Day 5
Advanced Shape Creation
5-6 hours

Objective: Confidently create complex yet controlled geometry using revolve, sweep, and loft.

üåÖ Morning: Revolve ‚Äî Cylindrical Parts (1.5 hours)

When to Use Revolve: Anything round ‚Äî knobs, bottles, vases, threaded parts, handles, cups.

Exercise 5.1 Custom Knob 
knob_diameter = 30 mm
knob_height = 20 mm
shaft_diameter = 6 mm
shaft_depth = 12 mm
grip_indent = 2 mm
  1. Sketch on XZ plane (side view)
  2. Draw HALF the knob profile (right side of centerline only)
  3. Include: flat top, curved grip section, base flare, center shaft hole
  4. Add a vertical construction line on the Y-axis as your revolve axis
  5. Create ‚Üí Revolve, select profile, select axis, 360¬∞
Key Insight: You only sketch HALF because revolve creates the full rotation.

‚òÄÔ∏è Midday: Sweep ‚Äî Following a Path (1.5 hours)

When to Use Sweep: Profiles that follow a curved path ‚Äî cables, handles, rails, tubes.

Exercise 5.2 Cable Clip 
clip_path_radius = 25 mm
clip_profile_width = 8 mm
clip_profile_height = 4 mm
cable_diameter = 5 mm
  1. Sketch the PATH on XY plane: an arc or curved line the clip will follow
  2. Create a new sketch on a plane PERPENDICULAR to the path start
  3. Sketch the PROFILE: the cross-section of the clip (rectangle with semicircle cutout for cable)
  4. Create ‚Üí Sweep, select profile, select path
  5. The profile "sweeps" along the path

üåÜ Afternoon: Loft ‚Äî Smooth Transitions (2 hours)

When to Use Loft: Connecting different shapes smoothly ‚Äî ergonomic grips, transitions, organic forms.

Exercise 5.3 Ergonomic Tool Handle 
handle_length = 100 mm
bottom_width = 25 mm
bottom_depth = 18 mm
middle_width = 22 mm
middle_depth = 28 mm
top_diameter = 18 mm
  1. Create sketch on XY plane: rounded rectangle bottom_width x bottom_depth
  2. Construct ‚Üí Offset Plane, 50mm up
  3. Create sketch: ellipse middle_width x middle_depth
  4. Construct ‚Üí Offset Plane, 100mm up from origin
  5. Create sketch: circle top_diameter
  6. Create ‚Üí Loft, select all three profiles in order
  7. Optional: add guide rails for more control over the transition shape
Exercise 5.4 Section Analysis
  1. Inspect ‚Üí Section Analysis
  2. Drag the section plane through your lofted handle
  3. Check for: thin walls, unexpected geometry, printability issues
  4. This is how professionals validate complex shapes

üåô Late Afternoon: Threads (1 hour)

For 3D Printing Threads:
  • Use MODELED threads (actual geometry), not cosmetic
  • Scale thread diameter up 2-5% for tolerance
  • Minimum M4 for FDM (smaller threads don't print well)
  • Print threads vertically for best results
Exercise 5.5 Threaded Bottle Cap 
bottle_od = 30 mm
bottle_id = 26 mm
thread_pitch = 3 mm
cap_height = 15 mm
thread_clearance = 0.3 mm
  1. Create bottle neck with external threads (Create ‚Üí Thread, check "Modeled")
  2. Create cap as separate component
  3. Cap internal diameter = bottle_od + thread_clearance*2
  4. Add internal threads to cap (same pitch, check "Modeled")
  5. Test fit with joint, then PRINT AND TEST

Day 6
Optimization & Organic Shapes
5-6 hours

Objective: Optimize designs for strength, weight, and print efficiency. Introduction to sculpting.

üåÖ Morning: Material Reduction & Strength Optimization (2 hours)

The Amateur Mistake: Adding material for strength.

The Professional Approach: Strategic geometry.

Optimization Strategies

StrategyMaterial SavedStrength Impact
Shell instead of solid60-80%Minimal loss if walls adequate
Ribs instead of thick walls50-70%Often INCREASES strength
Honeycomb infill areas40-60%Good for non-structural areas
Topology-inspired shapes30-50%Requires understanding load paths
Exercise 6.1 Redesign for Efficiency

Take your Day 3 enclosure and:

  1. Identify non-structural areas
  2. Add ventilation holes or weight-reduction pockets
  3. Replace any thick sections with ribbed sections
  4. Target: 30-50% material reduction
  5. Verify with Section Analysis that walls remain adequate

‚òÄÔ∏è Midday: Print Orientation & Layer Strength (1.5 hours)

Critical Concept: FDM parts are 50-80% weaker along layer lines than across them.
Exercise 6.2 Orientation Analysis

Take your Day 3 hook design:

  1. Identify the primary load direction
  2. Sketch three different print orientations
  3. For each, mark where layer lines would be
  4. Determine which orientation puts layers PERPENDICULAR to the load
  5. Document your reasoning

Print Orientation Decision Matrix

Part FeaturePreferred Orientation
Holes (need to be round)Hole axis vertical
ThreadsThread axis vertical
Snap fitsFlex direction across layers
Thin wallsWall plane vertical
OverhangsMinimize or face up

üåÜ Afternoon: Form Environment ‚Äî Organic Sculpting (2 hours)

When to Use Sculpt (Form Mode): Ergonomic grips, organic shapes, artistic designs, anything that doesn't fit boxes and cylinders.

Key Sculpt Tools

ToolWhat It Does

Exercise 6.3 Ergonomic Grip
  1. Create ‚Üí Form ‚Üí Box (this enters Form mode)
  2. Set divisions: 4 x 4 x 8
  3. Use Edit Form to pull and shape into a comfortable grip
  4. Enable Symmetry (Modify ‚Üí Symmetry ‚Üí Mirror - Internal)
  5. Subdivide areas needing more detail
  6. When satisfied, click "Finish Form"
  7. The T-Spline converts to a solid body
Exercise 6.4 Combining Sculpt with Solid
  1. Create a solid base with mounting features (screw holes, flat bottom)
  2. Create a sculpted grip section
  3. Use Modify ‚Üí Combine to join them
  4. Add fillets at the transition for smooth blend

Day 7
End-to-End Design Confidence
6-8 hours

Objective: Apply a repeatable workflow to solve any design challenge from blank canvas to printed part.

üîÑ The Professional Workflow (Use This Every Time)

1. DEFINE    ‚Üí What problem am I solving? What are the constraints?
2. PARAMETERS ‚Üí Create all dimensions as parameters BEFORE sketching
3. SKETCH    ‚Üí Fully constrained, origin-based, single-purpose sketches
4. MODEL     ‚Üí Logical feature order, components for separate parts
5. VALIDATE  ‚Üí Section analysis, interference check, wall thickness
6. OPTIMIZE  ‚Üí Reduce material, add ribs, consider print orientation
7. EXPORT    ‚Üí STL in mm, one file per component
8. TEST      ‚Üí Print, measure, adjust parameters, repeat

üéì Expert Instructor's Final Notes

The 3 Skills That Separate Amateurs from Professionals

After teaching hundreds of students, I've identified the three skills that most clearly separate those who "know Fusion" from those who can actually design:

  1. Parametric Discipline ‚Äî Professionals NEVER hard-code dimensions. Every number is a parameter with a meaningful name. This isn't about being fancy ‚Äî it's about being able to iterate quickly when (not if) requirements change.
  2. Constraint Thinking ‚Äî Amateurs dimension everything. Professionals constrain relationships. "These holes are always 10mm from the edge" is better than "this hole is at X=45mm." When the part size changes, the holes move correctly.
  3. Print-First Mindset ‚Äî Every design decision considers: "How will this print? Where are the layer lines? What's the weakest point?" This becomes automatic with practice.
